/**
 * Installation
 */
1. Enable Custom Clock Module admin/modules
2. Navigate admin/structure/block and click "Place Block" on required region
3. Select "Custom Clock" Block and update the fields such as Country, City and TimeZone and Save the block.

Note
----
By default TimeFormat will show like (7th Dec 2021 - 09:52 PM)
If you face issues with serviceNotFound issue. Update pathauto module.
